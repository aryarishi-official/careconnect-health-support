## Overview
CareConnect is a concept-level healthcare support platform designed for NGOs to manage patient support requests and volunteer registrations efficiently.

The platform includes a chatbot-driven interface and automated request summary generation to streamline operations.

---

## Tech Stack

### Frontend
- React (Vite)
- Vanilla CSS

### Backend
- Django
- Django REST Framework

### Database
- SQLite

### AI / Automation Feature
- Automated request summary generation for patient support
- Structured FAQ chatbot with guided routing

---

## Key Features

- Patient Support Request Form
- Volunteer Registration Form
- Chatbot with predefined options
- FAQ automation
- AI-generated request summary
- Request ID generation and display
- Tab-based request listing

---

## NGO Use Case

This system helps NGOs:
- Collect structured patient requests
- Automatically generate summaries for faster review
- Manage volunteer onboarding
- Provide quick FAQ responses through chatbot
- Track requests with unique IDs
